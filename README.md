# ماشین حساب ساده پایتون

یک برنامه ماشین حساب ساده که با زبان پایتون نوشته شده است. این پروژه برای انجام عملیات‌های پایه‌ای جمع و تفریق ریاضی است و قابلیت گسترش برای توابع پیشرفته‌تر را دارد. طراحی ساده و کاربردی آن، این برنامه را به نقطه شروع عالی برای مبتدیان پایتون و ابزاری مفید برای محاسبات سریع تبدیل کرده است.

## توضیحات

ماشین حساب پایتون یک برنامه سبک است که از عملیات اصلی ریاضی پشتیبانی می‌کند. این برنامه با پایتون خالص نوشته شده و به راحتی قابل فهم، تغییر و گسترش است. چه نیاز به محاسبات سریع داشته باشید و چه بخواهید با برنامه‌نویسی پایتون آشنا شوید، این ماشین حساب انتخاب مناسبی است.

## ویژگی‌ها

- **عملیات پایه**: جمع، تفریق.
- **قابلیت گسترش**: به راحتی می‌توانید با تغییر کد، توابع جدید اضافه کنید.

## نصب

### پیش‌نیازها
- پایتون نسخه ۳.۶ یا بالاتر بر روی سیستم شما نصب باشد.

### مراحل نصب
1. مخزن را کلون کنید:
   ```bash
   git clone https://github.com/MariVorzg372/my-calculator.git
   ```
2. به دایرکتوری پروژه بروید:
   ```bash
   cd my-calculator
   ```

3. ماشین حساب را اجرا کنید:
   ```bash
   python main.py
   ```

هیچ وابستگی خارجی نیاز نیست—فقط پایتون کافی است!

## نحوه استفاده

پس از اجرای برنامه، دستورالعمل‌های روی صفحه را دنبال کنید. از شما خواسته می‌شود مانند راهنما دو عدد وارد کنید. نتیجه بلافاصله نمایش داده می‌شود.



### سفارشی‌سازی
به راحتی می‌توانید با افزودن توابع جدید در کد منبع، ماشین حساب را گسترش دهید. به عنوان مثال، برای اضافه کردن عملیات توان، یک تابع جدید تعریف کرده و منو را به‌روز کنید.
